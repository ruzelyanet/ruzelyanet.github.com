Your safe with us!

Data security is the basic core of any community's development. We use the most up-to-date information system security standards. All ImRix accounts are securely protected by a multi-level security system, encryption system, and are backed up daily.

  • Secure Socket Layers (SSL). All accounts work using an SSL certificate issued by a recognised international certificate authority (Thawte).

  • Independent shared storage. All accounts work independently of each other thanks to Modern dockers containerisation technology.

  • Real-time backups. Real-time backups are performed on separate backup servers, which the system automatically switches to in the event of an accident. This seamless switching takes place without the user even noticing.
